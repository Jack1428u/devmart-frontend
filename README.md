# DevMart

La plataforma definitiva de comercio electrónico diseñada para desarrolladores y entusiastas de la tecnología. DevMart combina una experiencia de usuario ultra-minimalista con una arquitectura frontend rápida, moderna y de alto rendimiento.

## Características Clave

* **Diseño Minimalista Premium**: Una interfaz limpia, sin distracciones, enfocada en la facilidad de navegación y la velocidad de decisión de compra.
* **Experiencia de Usuario Fluida (UX)**: Micro-animaciones interactivas, estados de carga inteligentes mediante esqueletos (skeleton loaders) y transiciones instantáneas.
* **Carrito en Tiempo Real**: Gestión optimizada del flujo de compras con actualización del subtotal y validación de stock al instante.
* **Seguridad y Control**: Flujos de autenticación integrados con retroalimentación visual clara para el inicio y fin de sesión.
* **Diseño Responsivo Total**: Experiencia fluida adaptada perfectamente a cualquier pantalla, desde dispositivos móviles compactos hasta monitores de escritorio ultra-wide.

## Stack Tecnológico

* **React 19**: Aprovechando el compilador de React para un rendimiento de renderizado superior.
* **Vite**: Desarrollo ágil y bundling ultra rápido.
* **Tailwind CSS v4**: El motor CSS más avanzado del mercado, configurado con variables de diseño personalizadas y sin degradados para mantener la elegancia de la marca.
* **React Router v8**: Enrutamiento ultrarrápido sin recargas de página.


## Cómo Iniciar

### Requisitos Previos

Asegúrate de contar con Node.js y el gestor de paquetes **pnpm** instalados.

### Instalación de Dependencias

```bash
pnpm install
```

### Ejecutar en Modo Desarrollo

Levanta el servidor local con recarga rápida (HMR):

```bash
pnpm dev
```

### Construcción para Producción

Compila y optimiza el proyecto para su despliegue final:

```bash
pnpm build
```
